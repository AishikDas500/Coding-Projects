# Student Result Tracker

## Description
A Python program that reads a CSV file of student marks, allows the user to select students by roll number, calculates total marks and percentages, and updates the CSV file.

## Features
- Load CSV file
- Search student by roll number
- Calculate:
  - Total Marks
  - Top 5 Percentage
  - All 6 Percentage
  - Best 5 Percentage
- Save updated CSV

## How to Run
1. Install dependencies:
   pip install pandas

2. pip install pandas
   python Student_Result_Tracker.py
   
4. Enter file path and follow prompts

## Notes
- CSV must contain:
  - Roll No. (1st Row)
  - Name (2nd Row)
  - 6 subject marks (Main 5 subjects from start)
